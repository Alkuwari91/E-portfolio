Unit 1: Introduction to Machine Learning (ML)

Welcome to Week 1.

Machine learning (ML) has become an integral part of modern life. From online browsing to banking decisions, ML is used to make decisions and perform profiling. The availability of real-time big data facilitates the learning of ML algorithms, enabling machines to autonomously make more accurate decisions (Schwab & Zahid 2020).

This week, we will explore the past, present, and future of ML, the challenges and opportunities of using algorithms, and the interplay between big data analytics, ML, and artificial intelligence (AI).

This week you will learn about:

  • The past, present, and future of machine learning.
  • Challenges and opportunities in using machine learning algorithms.
  • The relationship between big data, machine learning, and AI.

On completion of this week, you will be able to:

  • Understand the role of machine learning in future industries.
  • Identify the skills required to become proficient in machine learning.
  • Recognize the potential pitfalls of machine learning and how to address them.

Unit 2: Exploratory Data Analysis

This unit focuses on techniques for understanding data before applying machine learning models. Exploratory Data Analysis (EDA) involves summarizing data sets, detecting patterns, and using visualization tools to gain insights. You will learn about libraries like Pandas and Matplotlib in Python to assist with EDA.

Unit 3: Correlation and Regression

This unit focuses on understanding the relationships between variables and building predictive models using regression techniques. Learn how correlation works and how to apply linear and logistic regression models.

Unit 4: Linear Regression with Scikit-Learn

Learn how to implement linear regression models using Python’s Scikit-Learn library. This unit will teach you how to prepare data, fit a model, and evaluate its performance using real-world datasets.

Unit 5: Clustering

Clustering is an unsupervised learning technique used to group similar data points together. This unit will cover popular clustering algorithms such as K-Means and DBSCAN.

Unit 6: Artificial Neural Networks

Learn about artificial neural networks (ANNs), which mimic the brain’s structure and functionality. This unit will introduce the basic concepts of ANNs, including activation functions, layers, and backpropagation.

Unit 7: Convolutional Neural Networks (CNNs)

In this unit, you will learn about Convolutional Neural Networks (CNNs), a specialized type of neural network designed for image and video recognition. Learn how to build and train CNNs for various applications.
